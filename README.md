# VBConn
# Lenguaje
Visual Basic .NET (VB.NET)

# Objetivo
Crear varios proyectos con distintos enfoques para la gestión de una cadena de conexión contra un motor de base de datos Microsoft SQL Server.

Para ello los proyectos deberán poder solicitar los datos de la conexión al usuario, probar la conexión contra el motor de BD, escribir la cadena conexión en un archivo de texto plano, leer la cadena desde el archivo, actualizar la cadena en el archivo y eliminar el archivo.

Cada proyecto servirá para un enfoque distinto de los mismos objetivos.
